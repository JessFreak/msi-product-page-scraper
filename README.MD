# MSI Product Page Scraper

This is a simple, lightweight web scraper designed to extract product details from the official MSI US Store. It is built with **Node.js** and **Playwright**.
The scraper runs in **headless mode**.

## Features
1. Extracts product details: title, brand, description, category tree, pricing (regular and sale), availability, specs, and images.
2. Saves extracted data into a clean, normalized JSON file at `output/product.json`.

## Getting Started

### Installation

1. Clone the repository:
```bash
  git clone https://github.com/JessFreak/msi-product-page-scraper.git
  cd msi-product-page-scraper
```

2. Install dependencies:

```bash
npm install
```

3. Install Playwright Chromium browser:

```bash
npx playwright install chromium
```

## Usage

To run the scraper, execute the following command:

```bash
npm run scrape
```

Once the process finishes, you will find the scraped data saved in `output/product.json`.
